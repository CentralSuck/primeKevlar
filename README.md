# primeKevlar

Installation:
1) Add primeKevlar to your resource folder
2) Import the sql file in your database
3) Add 'start primeKevlar' in your server.cfg

If you are using the ESX limit system, change in the sql file 'weight' to 'limit'

Depedencies:
- ESX
- skinchanger
